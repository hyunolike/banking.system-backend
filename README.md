🚀 **프로젝트 개요**

- 아래와 같이 크게 3가지 서버로 구성되어 있습니다.
- **🖥️ Scheduler Server** 는 지정된 시간에 자동으로 요청을 보내는 서버입니다.
- **🖥️ API Server** 는 받은 요청에 따른 알림 서비스 제공을 위한 서버입니다.
- **🖥️ Notification Service**는 네이버 클라우드에서 제공하는 Application Services입니다.

<br>

📌 **프로젝트 목표**

- 짧은 기간내에 완성할 수 있도록 적절한 프레임워크 부가 기능을 사용한다.
- 요구사항에 적절한 기술을 선택해 쉽게 개발을 진행한다.
- Github Actions을 이용해 배포 자동화를 구축한다.
